How it works

Paste. Convert. Import.

1

Paste your stat block

Text from any source — Monster Manual, homebrew PDFs, D&D Beyond, or a name and CR for the AI to build from scratch.

2

Get an import-ready file

The parser extracts every field — abilities, AC, HP, saves, skills, senses, spellcasting, actions, legendary actions — and outputs Foundry VTT JSON (dnd5e v4+) or Fantasy Grounds XML (2024 format).

3

Run the import macro

Copy the generated macro, paste it into Foundry's macro editor, and run it. The actor appears in your world — folders, items, and all.

🔑

AI features require a Claude API key

Name lookup, AI generation, image parsing, and magic item creation use the Claude API. Bring your own key — it's stored locally in your browser, never sent to our servers. Get a key at Anthropic →
